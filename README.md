# [F] ahmy >_

A high-end, brutalist, and kinetic personal portfolio built for the modern web. Designed to leave a lasting impression through seamless physics, micro-interactions, and dark cinematic aesthetics.

## 🌟 Live Demo
[thefahmy.vercel.app](https://thefahmy.vercel.app)

## 🚀 Tech Stack

- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS (Custom utility-first brutalist design)
- **Animations:** GSAP (ScrollTrigger, Parallax, Kinetic Typography)
- **Smooth Scrolling:** Lenis (Studio Freight)
- **Language:** TypeScript
- **Deployment:** Vercel

## ✨ Key Features

- **Kinetic Typography:** Custom scrolling text marquees and staggered character reveals.
- **Cinematic Page Transitions:** GSAP-powered curtain reveals that eliminate sudden page loads.
- **Magnetic UI Elements:** Physics-based buttons that react to cursor proximity.
- **Floating Cursor Previews:** Premium archive table with GSAP-tracked dynamic image previews on Desktop.
- **Ambient Lighting:** Subtle, high-performance cinematic glows that don't compromise 60fps scrolling.
- **Responsive Mastery:** Distinct layouts tailored for Desktop (Brutalist Grid) and Mobile (Fluid Touch).
- **Ascii Art Integration:** Developer-centric visual Easter eggs embedded directly into the UI.

## 🛠️ Getting Started

First, install the dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
